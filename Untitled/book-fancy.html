30.3.3 Google fonts

To change the appearance of your text with a new font, you need to:

  • Pick a Google font
  • Add it to your CSS file
  • Style specific text using CSS selectors
  • Make sure your CSS file is linked to _output.yml (and if youâve been following along and using the book skeleton, it already will be).

30.3.3.1 Select and import your Google fontâs script

  1. Go to https://fonts.google.com/ and pick a font for your bookâs main body text. Weâll choose Muli, but Lato and Source Sans Pro are other nice options.

30.3.4 Add a logo

Links are great ways to pepper your site with an accent color without it becoming too overwhelming. Letâs style the linksâ regular appearance and also what they will look like when hovered over.

30.3.6 Add a hero image

A hero image is âweb-speakâ for the large banner image that usually stretches across the page of many websites. We can sort of accomplish this in bookdown with a combination of both HTML and CSS. Youâll need:

  • An HTML file
  • CSS to style the image

Letâs get started:

30.3.6.1 Embed the image in an HTML file

  1. Create a new HTML file by going to File > New File > Text File.

  2. Paste these contents at the top of the file.

    <link href="style.css" rel="stylesheet">

</div>
